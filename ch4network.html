Chapter 4 Networking & The Internet
Term Definitions
Network Classifications
  • LAN: a local area network
  • MAN: a metropolitan area network
  • WAN: a wide area network
Network Models
  • Bus Network: network in which all machines are connected to a common communication line called a bus.
  • Star Network: one machine servers as a central focal point to which all the others are connected.
Protocols
  • CSMA/CD Ethernet: this protocol dictates that each message be broadcast to all the machines on the bus, each machine monitors all the messages but keeps only those addressed to itself.
  • CSMA/CA Wifi: policy that wireless networks adopt to try and avoid collisions rather than trying to detect them.
Combining Networks
  • Repeater: a device that passes signals back and forth between the two original buses without considering the meaning of the signals.
  • Bridge: similar, but more complex, to a repeater. It connects two buses, but it does not necessarily pass all the messages across the connection.
  • Switch:essentially a bridge with multiple connections, allowing it to connect several buses rather than just two.
  • Router: a device that handles connections between networkds to form an internet.
  • Gateway: the "point" at which one network is linked to an internet.
Client/Server Model
  • P2P (peer to peer): involves processes that provide service to and receive service from each other.
  • Distributed System: consists of software units that execute as processes on different computers, used by many modern software systems.
  • Cloud Computing: where huge pools of shared computers on the network can be allocated for use by clients as needed, is latest trend in distributed systems.
Internet Addressing
  • Tier 1: consists of very high-speed, high-capacity, international WANs. Thought to be backbone of the Internet.
  • Tier 2: tend to be more regional in scope and less potent in their capabilities.
  • Tier 3: independent internet, operated by a single authority that is in the business of supplying Internet access to individual homes and businesses.
IP Address
  • ICANN: a non-profit corporation established to coordinate the Internet's operation.
  • DNS: name servers used collectively as an Internet-wide directory system.
  • TLD (Top-level domain): suffix used to reflect the domain's classification. ex. mu.edu
  • Country-code TLDs: two-letter TLDs for each specific country such as .au for Australia.
Internet Protocols
  • FTP: an application for accessing another computer from a great distance used the Telnet protocol, or later, the Secure Shell protocol.
  • HTTP: handles more sophisticated webservers and browsers.
  • SMTP: defines a way that two computers on the network may interact when transmitting an email message from one host to the other.
World Wide Web
  • URL: uniform resource locator, a unique address given to each document on the Web.
  • Client-side activities: activities performed by a client, such as a browser.
  • Server-side activities: activities performed by a server, such as a webserver.
Internet Four-Layer Software
  • Application Layer: where the message typically orginates in origin and where message is received in final destination.
  • Transport Layer: receives message from application layer, chops messages into packages in origin and collects packages and reassembles message in final destination.
  • Network Layer: assigns intermediate address to each package in origin and detects that package has reached its final destination in final destination.
  • Link Layer: transfers packet in origin and receives packet in final destination.
Internet Is:
  • Packet-Switching Network: data is divided into small segments known as packets.
  • Fault Tolerant: the ability of a system to continue functioning despite failures.
  • Redundant: the many possible pathways between routers.
Cybersecurity
  • Malware: malicious software that attacks the computer.
  • Virus: software that infects a computer by inserting itself into programs that already reside in the machine.
  • Worm: an autonomous program that transfers itself through a net-work, taking up residence in computers and forwarding copies of itself to other computers.
  • Trojan Horse: a program that enters a computer system disguised as a desirable program that is willingly imported by the victim. Inflicts harm.
  • Spyware: form of malicious software that collects information about activities at the computer on which it resides and reports information back to the instigator of the attack.
  • Phishing: a technique of obtaining information explicitly by simply asking for it.
  • DDoS Attack: the process of overloading a computer with messages.
Protection
  • Firewall: a program thats primary prevention technique to filter traffic passing through a point in the network.
  • Spam Filter: firewalls designed to block unwanted email.
  • Proxy Server: a software unit that acts as an intermediaryy between a client and a server with the goal of shielding the client from adverse actions of the server.
  • Antivirus Software: used to detect and remove the presence of known viruses and other infections.
  • Cryptography: the study of sending and receiving messages securely in the presence of adversaries.
  • Public Key Encryption: involves techniques by which the encryption and decryption algorithms are known, and one of the keys is known, but the encoded message remains secure.
